# 📸 YouTube Snapshot

A lightweight, high-performance browser extension that captures lossless, native-resolution snapshots of YouTube videos and Shorts with zero lag.

## ✨ Features
* **Zero Lag:** Uses asynchronous blob generation to ensure even 4K/8K videos never drop a frame when you take a snapshot.
* **Shorts Integration:** Seamlessly integrates into the YouTube Shorts vertical action bar.
* **Smart Naming:** Automatically names your downloaded images using the video's title and current timestamp.
* **Keyboard Shortcuts:** Simply press `S` to instantly grab a frame without reaching for your mouse.
* **Native Aesthetic:** Matches YouTube's modern UI language flawlessly (tooltips, icons, and hover states).

## 🚀 Installation

*This extension is currently pending review on the Chrome Web Store. For now, you can load it manually:*

1. Clone or download this repository.
2. Open your browser and navigate to the Extensions page (e.g., `chrome://extensions/` or `edge://extensions/`).
3. Enable **Developer mode**.
4. Click **Load unpacked** and select the folder containing this repository.
5. Open any YouTube video and click the camera icon (or press `S`)!
